# Arnav Chavan — CV

LaTeX source for my academic CV. Compiled PDF: [`CV.pdf`](CV.pdf).
Website: [arnav0400.github.io](https://arnav0400.github.io/)

## Build
```bash
cd src
lualatex cv.tex && lualatex cv.tex
```
Requires: `datetime2`, `ulem`, `fontawesome`, `tcolorbox`, `pagecolor`, `fullpage`, `luainputenc` (+ standard TeX Live).

Template adapted from [Just-Another-Research-CV](https://github.com/SebastinSanty/Just-Another-Research-CV).
